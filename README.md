A simple Flutter application for tracking points between Team A and Team B. The app allows adding 1, 2, or 3 points to each team and resetting the score. Designed as a beginner-friendly project to practice buttons, layout widgets, and basic state management

Features :

Add 1, 2, or 3 points for each team
Real-time score update
Clean UI using Rows, Columns, and Divider
Reset button to clear the score
Great practice for StatefulWidget and setState
